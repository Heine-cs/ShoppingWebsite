# OllieShop 簡易上手的購物平台!!

[![OllieShopLogo](https://i.imgur.com/VKfLYLV.png)](http://ec2-52-199-224-131.ap-northeast-1.compute.amazonaws.com)

### 簡介  
Ollie Shop是一個提供消費者與商家進行買賣的電子商務平台，平台從使用者需求出發，重視用戶體驗，開發聚焦在讓用戶都能無痛使用快速上手，商家能即時刊登商品，提升商品周轉率，平台也能協助商家完成行銷佈局、廣告投放，借平台之手經營並提升店家形象，使得商品的曝光度更高，也接納多樣化的個性商品販售，透過完善的分類流程，迅速協助消費者挑選出符合需求的商品；

為了建立無障礙的溝通管道，降低買方賣方溝通成本與促使客戶滿意度的提升，平台也提供買家賣家即時性溝通，買賣流程最佳化處理，並且能對於不當的行為提出檢舉，藉由平台進行對應的懲處，此外敏捷的流程也是開發中不斷優化的方針之一，即使對於網站操作不熟悉的你，也能藉由平台的協助順利下單、安心收貨；

Ollie Shop 購物平台開發始終基於服務客戶的使命感，也正是因為視用戶需求為己任，緊緊守護用戶真心的每份回饋，透過觀察用戶群體的共識持續優化系統，未來更期許能與商家、消費者一併成長，開發出更完善的功能，藉用戶意見反哺網站、敏捷開發也是系統營運的重要方針，站長認為只有追求卓越，才能爭取成功，這便是Ollie Shop初始架站所信奉的圭臬，最後也萬分感謝使用網站用戶間給予的支持，您的支持才是我進步的最大動力。

### 功能描述  
- 零售商能為自己的賣場彈性規劃多元的支付、運送方式。  
- 上架商品與訂單管理，第一時間掌握最新消費者下單消息，迅速買賣。  
- 透過選取商品大類來篩選商品，提升購物的體驗，迅速搜尋到您想要的商品。  
- 賣場搜尋商品功能，讓你透過關鍵字迅速定位並滿足你迫切的需求。  
- 同時對應四種不同身分的用戶，分別是消費者、業者、管理員、訪客，每種用戶都有存在的意義，與對應的功能可使用。  
- 免登入就能使用簡易直覺的購物車功能，使你能舒適悠哉的在賣場收入商品。  
- 透過自動化計算商品與折價券後的金額，能夠輕易地控制您的預算，保障您荷包的健康程度。  
- 訂單歷史紀錄與折價券管理功能，保留您過去的購買組合，讓您想再次下同性質訂單也免煩惱紀錄的問題。

### 外部實體 
1. 訪客  
訪客能於Ollie Shop篩選並瀏覽商品，並透過Search Bar下關鍵字搜尋理想的商品，在尚未註冊成會員的情況下能將商品加入購物車，能在公佈欄了解管理員發布的最新公告，其餘與商城相關的進階功能則須透過註冊，獲取身分後繼承所註冊身分組擁有的功能。  
2. 消費者  
消費者能於Ollie Shop登入畫面輸入帳密，登入後具有的功能:透過對商品的下單，完成訂單功能、透過訂單管理的功能查詢商品購買紀錄、領取管理員發布的折價券並應用訂單之上、違規事件的提交、在後台中更改個人資料…等功能。  
3. 商家  
商家能於Ollie Shop登入畫面輸入帳密，登入後具有的功能:商品的上下架與管理、維護收款與運送方式、獨有的商店前台供消費者選購自家商品、後台更改個人資料…等功能。  
4. 管理員  
管理員能於Ollie Shop的後台通過輸入正確的帳號與密碼，即可登入系統管理後台，能同時支援多個管理員登入，具有系統最高權限，主要負責賣場違規事件、發布公告、管理商品類別等…功能。
